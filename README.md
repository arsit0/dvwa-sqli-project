# DVWA SQL Injection 入门项目

## 项目简介
这是一个基于 DVWA（Damn Vulnerable Web Application）的 Web 安全入门小项目。我在本地搭建了 DVWA 环境，并完成了 SQL Injection 的基础复现和简单的信息提取实验。

## 项目环境
- Windows
- XAMPP
- DVWA
- 安全等级：low

## 项目内容
本项目主要完成了以下内容：
1. 搭建 DVWA 本地练习环境
2. 在 SQL Injection 模块中完成正常查询测试
3. 完成基础 SQL 注入复现
4. 尝试进行信息提取型注入
5. 查看漏洞源码并分析漏洞成因

## 实验结果
- 正常输入时，页面只返回单个用户信息
- 注入输入后，页面返回了多条记录，说明查询逻辑被改变
- 在进一步实验中，可以读取数据库中的更多信息，说明 SQL 注入不仅能绕过查询条件，还可能造成敏感数据泄露

## 我学到了什么
- SQL 注入和数据库查询有关
- 用户输入如果没有被安全处理，就可能影响 SQL 语句逻辑
- Web 安全漏洞很多都和输入校验和后端处理方式有关
- 仅靠简单过滤特殊字符并不可靠，更重要的是参数化查询
